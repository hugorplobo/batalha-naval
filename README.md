## Como executar

O projeto usa o [Poetry](https://python-poetry.org/) para o gerenciamento de dependências e ambientes virtuais. Para instruções de como instalá-lo acesse [aqui](https://python-poetry.org/docs/#installation).

```bash
  $ poetry install
  $ poetry run pytest
  $ poetry run start
```
